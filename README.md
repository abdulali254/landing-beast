## ğŸš€ Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/landing-beast.git
   cd landing-beast
   ```

2. **Install dependencies**

   ```bash
   # Using Bun (recommended)
   bun install

   # Using Yarn
   yarn install
   ```

3. **Start the development server**

   ```bash
   # Using Bun (recommended)
   bun dev

   # Using Yarn
   yarn dev
   ```

4. **Open your browser**
   Visit [http://localhost:3000](http://localhost:3000)
